Machine learning project that predicts crop yield based on soil nutrients and environmental conditions.

## Models Used
- XGBoost Regressor

## Features

- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## Output

## How to Run

Install dependencies:
pip install -r requirements.txt

Run training:
python crop_yield_predictor_.py

## Dataset
Place the dataset inside:
data/Crop_recommendation (1).csv
